## Strings

Practice tasks covering string manipulation in Python.

| File | Description |
|------|-------------|
| `rgb_encoder.py` | Encodes RGB values (0–255) into hex color code (#RRGGBB) |
| `case_validator.py` | Checks if a string consists entirely of uppercase letters |
| `string_methods.py` | Showcase of built-in string methods: count, startswith, endswith, zfill, center, strip |
